# Frontend Design Pro - Demo Showcase

11 production-ready HTML/CSS demos showcasing distinct aesthetic directions. Each demo is a single self-contained file with no framework dependencies.

## Style Gallery

| # | Style | Key Techniques |
|---|-------|----------------|
| 01 | **Minimalism & Swiss** | Grid systems, massive typography, asymmetric layout |
| 02 | **Neumorphism** | Extruded elements, multi-layer shadows, press animations |
| 03 | **Glassmorphism** | Animated mesh gradients, frosted glass, backdrop blur |
| 04 | **Brutalism** | Thick borders, hard shadows, masonry grid |
| 05 | **Claymorphism** | Inflated 3D clay, marshmallow shapes, candy pastels |
| 06 | **Aurora / Mesh Gradient** | Breathing blobs, floating glass overlays |
| 07 | **Retro-Futurism / Cyberpunk** | Neon glow, CRT scanlines, glitch effects |
| 08 | **3D Hyperrealism** | Realistic textures, cinematic lighting, physics motion |
| 09 | **Vibrant Block / Maximalist** | Clashing RGB blocks, snap hover effects |
| 10 | **Dark OLED Luxury** | Absolute black + gold, spotlight cursor |
| 11 | **Organic / Biomorphic** | Morphing blobs, wavy dividers, nature palette |

## Quick View

Open `demos-v02/index.html` in your browser to see all 11 styles in a single showcase page.

## Usage with Visual Workbench

These demos serve as reference implementations for the [Visual Workbench](../Visual-Workbench/) AI slide generator. The `frontend-design-pro` skill teaches the AI to generate slides matching these aesthetic standards.
